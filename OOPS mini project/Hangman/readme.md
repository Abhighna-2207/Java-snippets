# Hangman Game

Hangman is a classic word-guessing game where the player attempts to guess a hidden word by suggesting letters within a certain number of guesses. This Java implementation of Hangman provides a console-based experience that is simple yet entertaining.

## Features

- Interactive console gameplay with prompts for user input.
- Randomly selected words from a predefined list.
- Visual representation of the hangman as incorrect guesses accumulate.
- Ability to play multiple rounds.

## Getting Started

### Prerequisites

- Java Development Kit (JDK) installed on your machine.
- An IDE (like IntelliJ IDEA, Eclipse, or NetBeans) to run the code.
